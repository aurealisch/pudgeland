# Pudgeland

🔥 Pudgeland - Цей сервер, можливо, коштував усіх життєвих заощаджень кожному, хто мав честь побувати на ньому, але наша теорія полягає в тому, що воно того не варте. Від жалюгідних RP-серверів, до найпрекрасніших проєктів всієї Discord'у і Minecraft'a, цей величний сервер укрив під своїми балюстрадами сотні талановитих бактерій і не дуже багато талановитих людей
