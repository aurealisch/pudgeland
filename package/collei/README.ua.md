# Alhaitham

> Це частина `Pudgeland 💖 Open Source` екосистеми

Неофіційна [_The Dog API_](https://thedogapi.com) оболонка для Python

## 📦 Пакети

### 🐍 PyPI

```sh
pip install alhaitham
```

## 🔎 Приклади

```py
import alhaitham

client = alhaitham.Client()

for _ in range(5):
  print(client.images.search())
```
