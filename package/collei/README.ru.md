# Alhaitham

> Это часть `Pudgeland 💖 Open Source` экосистемы

Неофициальная [_The Dog API_](https://thedogapi.com) оболочка для Python

## 📦 Пакеты

### 🐍 PyPI

```sh
pip install alhaitham
```

## 🔎 Примеры

```py
import alhaitham

client = alhaitham.Client()

for _ in range(5):
  print(client.images.search())
```
