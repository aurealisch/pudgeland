# Nahida

> Це частина `Pudgeland 💖 Open Source` екосистеми

Неофіційна **[Waifu.pics](https://waifu.pics)** API оболонка для Python

## 📦 Пакети

### 🐍 PyPI

```sh
pip install nahida
```

## 🔎 Приклади

```py
import nahida

client = nahida.Client()

print(client.nsfw.search("neko"))

print(client.sfw.search("awoo"))
print(client.sfw.search("bite"))
```
